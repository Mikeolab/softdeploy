#### Step 3: Create a Real Test
**"Let me create a real e-commerce test in 2 minutes..."**

1. **Select Test Type**: "API Testing"
2. **Add Step 1**: "Login to e-commerce site"
   - Method: POST
   - URL: https://api.example.com/login
   - Body: {"email": "user@example.com", "password": "password"}
   - Validation: Status 200, Extract session token

3. **Add Step 2**: "Search for product"
   - Method: GET
   - URL: https://api.example.com/search?q=laptop
   - Headers: Authorization: Bearer {session_token}
   - Validation: Status 200, Response time < 500ms

4. **Add Step 3**: "Add to cart"
   - Method: POST
   - URL: https://api.example.com/cart/add
   - Body: {"product_id": "123", "quantity": 1}
   - Headers: Authorization: Bearer {session_token}

#### Step 4: Execute Real Test
- Click "Execute Test Suite"
- Show real-time WebSocket updates
